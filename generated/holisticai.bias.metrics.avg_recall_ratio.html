holisticai.bias.metrics.avg_recall_ratio

holisticai.bias.metrics.avg_recall_ratio(g_min, g_maj, mat_pred, mat_true, top=None, thresh=0.5)

Average recall ratio

This function computes the ratio of average recall (over users) on minority and majority group.

Parameters

  • g_min (numpy array) – Minority class vector.

  • g_maj (numpy array) – Majority class vector.

  • mat_pred (numpy ndarray) – Matrix with shape (num_users, num_items). A recommender score (binary or soft pred) for each user,item pair.

  • mat_true (numpy ndarray) – Matrix with shape (num_users, num_items). A target score (binary or soft pred) for each user,item pair.

  • (optional) (thresh) – If not None, the number of items that are shown to each user.

  • (optional) – Threshold in (0,1) range indicating value at which a given item is shown to user (if top is None).

Returns

Ratio of average recalls : AVg_recall_min/AVg_recall_maj

Return type

float
